# muc-challange

## Backend

### Setup

#### 1. install npm

Ubuntu:

```sh
# install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# install npm version 22
nvm install 22
nvm use 22
nvm alias default 22
```

#### 2. install packages

```sh
npm install
```

### Run Backend Service in Dev enviorement

```sh
npm run dev
```
